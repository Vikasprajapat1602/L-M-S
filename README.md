# 📚 Library Management System (LMS) :

A Django-based web application designed to help libraries manage books, students, and circulation activities efficiently.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🚀 Features :

- 📚 Book Management (add, update, delete books).
- 📖 Issue & Return Books (with availability check).
- 🧮 Fine Calculation (based on late returns).
- 📅 Due Date Reminders.
- 📊 Admin Dashboard.
- 🧾 Detailed Book Issue History.
- 🧑‍🎓 Student Portal (view issued books, return status, due dates).
- 
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🛠️ Tech Stack :

- Backend: Django (Python).
- Frontend: HTML, CSS, Bootstrap.
- Database: SQLite.
- Tools: Git, GitHub, VS Code.
